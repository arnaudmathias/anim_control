# anim_control

Tinkering with 3D animation blending

Usage
-----
`./anim_control`

### Keymap:  
**WASD**  - move around  
**F**     - toggle fullscreen  
**,**     - decrement timestep  
**.**     - increment timestep  
**I**     - toggle debug info HUD  

Build
-----
```
git clone --recursive https://github.com/indiedriver/anim_control
cd anim_control
cmake .
```
